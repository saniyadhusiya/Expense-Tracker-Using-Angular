# Expense Tracker Using Angular

Built a responsive Expense Tracker application using Angular with CRUD operations, Reactive Forms, Angular Material, Signals, and JSON Server API integration.

## Features

- Add, Edit, Delete, and View Expenses
- Reactive Forms with Validation
- Angular Material UI
- Standalone Components
- JSON Server REST API Integration
- Responsive Design
- Reusable Components
- Professional Dashboard UI

## Technologies Used

- Angular
- TypeScript
- Angular Material
- Reactive Forms
- JSON Server
- Bootstrap
- HTML5
- CSS3

## Run the Project

1. Install dependencies

```bash
npm install
```

2. Start JSON Server

```bash
npx json-server db.json --port 3000
```

3. Run Angular

```bash
ng serve
```

4. Open

```
http://localhost:4200
```
